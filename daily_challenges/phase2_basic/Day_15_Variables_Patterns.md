### Step 1: Create a Pattern with Variables
- Open FMSLogo and type `CS` to clear the screen.
- Try this code:
  ```
  MAKE "length 20
  MAKE "angle 90
  REPEAT 4 [
    FD :length
    RT :angle
    MAKE "length :length + 10 ; Increase length for each side
  ]
  ```
- Observe how the pattern grows with each iteration!
